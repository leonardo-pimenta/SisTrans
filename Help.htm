Arquivo:
 

      Possui três opções:

  • CGS - Esta opção possibilita Listar multas a partir de uma data e Listar multas com requerimento. Só estará disponível se logar com a senha do CGS.

  • Logoff - O usuário corrente efetua a desconexão do sistema para uma mudança de usuário ou seu encerramento.

 

  • Sair - Encerra o sistema.

voltar
 
 

Cartão:
 
      Possui as seguintes opções:

  • Veículo - Emite um cartão a partir de um veículo específico.

  • Proprietário - Emite cartão de:

    • Pessoa - Informando o CPF.Será impresso todos os veículo pertencentes aquela pessoa.

    • OM - Informando o código da OM, sendo que podem ser emitido dois tipos de cartão de OM , o primeiro é a partir de um veículo específico, já que uma OM pode ter vários veículos, a segunda é a emissão de um cartão sem veículo só com os dados da OM. Toda OM tem direito a ter esse cartão para dar a quem for de direito.

    • Fornecedor- Informando o CNPJ.Será impresso todos os veículos do CNPJ digitado em cartões separados.

    • Provisório - Cartão que tem prazo de validade e só pode ser destinado à pessoas. Lembrando que para se emitir um cartão provisório é necessário cadastramento prévio da pessoa e do veículo.

voltar
 
 
Multar.:
 
      Possui as seguintes opções:

  • Multar - Cadastrar o veículo multado.

  • Requerer ao CGS - Informações do requerente que justificam o pedido de anulação da infração.

  • Multas ao DETRAN - A princípio todas as multas estão indeferidas, neste caso, se dentro de 72 horas o CGS não der deferido, elas seguem automaticamete para o DETRAN.

voltar
 
Relatórios: 
      Possui as seguintes opções:

  • Multas com requerimento - Listagem de todas as multas que possuem requerimento,deferidas,indeferidas ou que aguardam definição do CGS.

  • Multas sem requerimento - É necessário que as multas tenham requerimento para serem deferidas pelo CGS, se a mesmas não possuirem, dentro de 72 horas seguem ao DETRAN, já que permanecem na situação de indeferidas.

  • Lista de Deferidos - Somente as que foram deferidas pelo CGS.

  • Listar indeferidos - Lista todas as multas que foram indeferidas pelo CGS.
  • Listagem de Cartão - Listagem por:
    • Nome do proprietário- Lista todos os cartões de uma determinada pessoa.
    • Nº do cartão - Lista um cartão específico.
  • Listagem de veículo - De uma determinada(o):

  • Pessoa

  • OM

  • Fornecedor

  • Motorista/Fornecedor

  • Todos

  • Unitário

 Manutenção: 

     Possui as seguintes opções:

  • Usuário - Inclusão, alteração ou exclusão de usuário.
  • Marca Modelo - Inclusão, alteração ou exclusão de uma nova marca ou modelo de veículo.
  • Especialidade - Inclusão,alteração ou exclusão de uma especialidade.
